## 🚀 Sobre o Projeto (Meu Portfólio)

Este repositório contém o código fonte do meu portfólio pessoal, com foco em uma UI moderna, limpa e de alta performance. O objetivo é demonstrar habilidades práticas de engenharia de software no front-end em um ecossistema produtivo contemporâneo.

### 🛠 Tech Stack e Ferramentas Utilizadas
- **Framework:** [Next.js](https://nextjs.org) (App Router) + React 19
- **Linguagem:** TypeScript
- **Estilos & Componentes UI:** 
  - [Tailwind CSS v4](https://tailwindcss.com/)
  - [shadcn/ui](https://ui.shadcn.com/) e [@base-ui/react](https://base-ui.com/)
  - Suporte a temas dinâmicos via `next-themes`
- **Animações & Motion:** [Motion](https://motion.dev/)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **Qualidade de Código:** 
  - Linting e Formatação por [Biome](https://biomejs.dev/)
  - Princípios de Clean Architecture & boas práticas do ecossistema React/Next.js.

### ⚙️ Como Executar o Projeto Localmente

1. Clone este repositório:
```bash
git clone https://github.com/Williamlp-dev/portfolio.git
cd portfolio
```

2. Instale as dependências (recomendado usar `bun`):
```bash
bun install
```

3. Inicie o servidor de desenvolvimento:
```bash
bun dev
```

4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
